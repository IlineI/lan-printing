关于项目

本项目是一个内网打印及扫描服务软件，旨在解决因为各种原因（如驱动兼容性、网络问题、系统差异等）导致无法连上共享打印机的困扰。不再依赖繁琐的驱动安装和共享配置，只需在连接打印机的电脑上安装并使用管理员权限运行软件，同一局域网下的其他设备即可通过该主机电脑提供的IP端口进入Web界面，轻松访问和使用目标电脑的打印和扫描功能，完全绕过共享打印的各种限制和兼容性问题。

使用说明

建议：
需确保文件路径不含中文，若带中文则可能导致程序无法运行。
如遇无法运行，可能是权限不够或被防火墙/安全软件拦截，请关闭防火墙和安全软件后使用管理员权限重新运行
取消打印后马上扫描可能导致打印机/扫描仪被锁定，请人工重启打印机/扫描仪。

  1. 在接打印机的目标电脑上创建文件夹，将软件放入（最好win10，电脑名、文件夹和软件名都需是英文）
  2. 右键exe文件，使用管理员权限打开，托盘栏会生成图标，短暂显示控制台弹窗
  3. 控制台5秒后隐藏，等待控制台隐藏即可。若5秒内按了任意键则会保留显示控制台。
    可通过右键托盘栏图标，控制其显示和隐藏，关闭控制台则会关闭软件
  4. 右键托盘栏图标，会展示IP端口，其他设备在浏览器地址栏输入该IP端口即可进入可视化网页，在网页进行打印和扫描操作
  5. 右键托盘栏图标，可安装二维码生成插件，做成二维码分发给局域网的其他人使用
  6. 其他设备通过该IP端口访问网页后，通过【打印】和【扫描】标签页切换功能：
    • 打印功能：选择/拖拽文件上传，设置打印参数后提交即可
    • 扫描功能：选择扫描仪，设置扫描格式，点击扫描按钮即可
  7. 上传后可在队列中删除未打印的文件，随时取消
  8. 扫描完成的文件可直接预览、下载或重新打印，支持点击"清空队列"按钮一键清空所有扫描文件
  9. 上传的文件会保存在主机的【uploads】文件夹里，若不主动删除，则会在10分钟后自动删除
  10. 扫描的文件会保存在主机的【scanned_files】文件夹里，若不主动删除，则会在30分钟后自动删除
  11. 扫描仪被锁定时，会自动强制释放，无需重启主机电脑或手动重启扫描仪，等待1-5分钟即可。
    若仍未释放，请人工重启扫描仪
  12. 所有操作均有提示，界面直观，无需额外说明
  13. 扫描和打印操作会自动避免设备冲突
  14. 网络相关设置请在托盘菜单【网络配置】中操作（必须管理员权限）
  15. 支持静态IP设置：选择"设置当前IP为静态"可固定当前IP地址
  16. 智能打印优化：程序会自动优化文档边距，无需手动调整
  17. 日志自动维护，无需手动清理（每天下午3点，会自动清理7天前的记录）
  18. 控制台自动维护，无需手动清理（超过1000行自动清理）

扫描功能说明

支持的扫描仪类型

  • WIA兼容扫描仪
  • 多功能一体机
  • USB扫描仪

不支持的扫描仪类型及考虑

  • TWAIN专用扫描仪：老旧设备，用户基数很小
  • 专业扫描仪：企业级设备，有专用软件
  • 网络扫描仪：大多有自己的Web界面
  • 跨平台扫描仪：主要是兼容性问题，非核心需求

扫描参数与输出

  • 分辨率：自动尝试300 DPI
  • 扫描模式：彩色扫描模式
  • 输出格式：PNG（推荐，最佳质量）、JPEG、PDF
  • 静默扫描：完全后台执行，无弹窗干扰
  • 失败处理：扫描失败时提供详细错误信息
  • 兼容性提示：如确实安装了扫描仪但扫描失败，请在目标电脑上进行扫描操作

其他

  • 支持 Win7/Win10/Win11 系统
  • 技术栈：Python 3.7.9 + Flask + Windows API + COM组件 + WIA扫描接口
  • 若使用源码运行，请解压【poppler.rar】压缩文件到项目根目录，用于处理PDF文件（仅适配老旧打印机需要）
  • 详细功能和更新请见托盘菜单或网页界面
  • GitHub 仓库：https://github.com/a937750307/lan-printing

依赖安装命令

全部复制粘贴到命令行即可：

pip install flask flask_cors werkzeug requests requests_toolbelt pysnmp pystray pillow comtypes pywin32 wmi waitress certifi chardet win32print win32api win32com ctypes winreg pystray pillow PIL Pillow pdf2image

建议在打包前先执行上述命令，确保所有依赖齐全。

打包建议使用 PyInstaller 5.x 及以上版本，推荐命令：

pyinstaller --clean --noconfirm print_server2.6.spec

如需自定义依赖或资源，请编辑 .spec 文件的 hiddenimports 和 datas 部分。
