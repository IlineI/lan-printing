## 🚀 怎么用？（3步搞定）

### 第一步：安装软件
- **简单方式**：下载exe文件，双击运行就行（建议右键"管理员运行"）
- **技术方式**：有Python环境的可以下载源码运行

### 第二步：启动服务
- 运行后看到托盘有个小图标就说明成功了
- 软件会显示一个网址，比如：http://192.168.1.100:5000
- 如果有问题，软件会告诉你缺什么，按提示下载安装就行

### 第三步：开始打印
- 用手机或其他设备打开刚才的网址
- 选择打印机和打印设置
- 把文件拖进去或点击选择文件
- 点"上传并打印"就完事了！

### 💡 小贴士
- **托盘图标很有用**：左键打开网页，右键有各种设置
- **端口被占用了**：右键托盘图标→服务设置→更改端口，保存后需要手动重新运行程序
- **想要开机自启**：右键托盘图标→开机自启→开启
- **网络有问题**：右键托盘图标→网络配置→按需设置

## ❓ 遇到问题怎么办？

### 软件打不开？
**症状**：双击exe没反应，或者提示缺少dll文件  
**解决**：下载安装VC++运行库（软件会提示下载地址），安装后重启电脑

### 找不到打印机？
**症状**：网页里显示"未检测到可用打印机"  
**解决**：
- 检查打印机是否开机、连接正常
- 确认打印机驱动已安装
- 点网页里的"刷新"按钮重新检测

### 网页打不开？
**症状**：输入地址后无法访问  
**解决**：
- 检查Windows防火墙，允许程序通过
- 尝试用127.0.0.1替代IP地址
- 右键托盘图标→更改端口，换个端口后手动重新运行程序

### 杀毒软件报毒？
**症状**：杀毒软件阻止运行  
**解决**：添加到白名单，或者从官方GitHub下载最新版本

### 更改端口后程序没反应？
**症状**：更改端口后程序还是使用旧端口  
**解决**：端口配置已保存，但需要手动重新运行程序才能生效。退出程序后重新双击exe文件启动即可

### 点击选择文件没反应？
**症状**：点击上传区域没有弹出文件选择框  
**解决**：
- 更新浏览器到最新版本
- 清除浏览器缓存
- 关闭广告拦截插件
- 直接拖拽文件到上传区域

### 想要横版打印或自定义页码？
**现实**：这个软件主要解决"能打印"的问题，高级排版功能有限  
**建议**：用Word、WPS等软件先排好版，设好参数，再上传打印
