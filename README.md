# 核心功能说明
**1. 手机直接打印，无需安装APP**
- 手机拍照后直接打印：拍个身份证、文档照片，马上就能打印出来
- 微信文件即开即印：微信收到的文件，用浏览器打开网址就能打印，不用传来传去
- 零安装使用：任何有浏览器的设备都能用，iPhone、安卓、iPad统统支持

**2. 电脑打印操作更简单**
- 拖拽打印：直接把文件从桌面拖到网页里，点一下就打印，比传统方式快3倍
- 全格式支持：PDF、Word、Excel、PPT、图片、纯文本文件都能打印
- 批量打印：一次选择多个文件，自动排队打印，不用一个个点

**3. 部署简单到极致**
- 一台电脑搞定：只需要在连打印机的电脑上装这个软件就行了
- 其他设备零配置：家里所有设备输入一个网址就能用，不用每台都装软件
- 托盘管理：右键小图标就能改端口、调网络，所有设置都在这里

**4. 智能错误处理和修复**
- 环境检测：自动检查缺少的运行库，直接给下载链接，小白也能解决
- 打印机诊断：检测不到打印机会告诉你具体原因和解决步骤
- 端口智能处理：端口被占用会提示解决方案，支持手动更改端口

**5. 打印参数设置专业全面**
- 智能打印机识别：自动识别你的打印机支持哪些功能，不支持的选项直接隐藏
- 全参数控制：打印机选择、打印份数、单双面、纸张大小、打印质量随你调
- 虚拟打印机过滤：自动过滤PDF虚拟打印机，只显示真正的物理打印机

**6. 网络配置傻瓜式操作**
- 自动IP分配：支持DHCP自动获取IP，也可以手动设置固定IP
- 网络故障诊断：有网络问题会一步步教你排查，从防火墙到端口全覆盖
- 端口冲突处理：检测到端口被占用会提示可用端口，支持手动更改

**7. 文件管理自动化**
- 智能清理：上传的文件10分钟后自动删除，不占用磁盘空间
- 大文件支持：支持上传大文件，有实时进度显示，不会卡死
- 操作日志：所有打印操作都有详细日志，出问题能快速定位
- 支持预览：打印前可预览内容，确保正确

**8. 用户界面体验优化**
- 响应式设计：手机、平板、电脑屏幕都能完美适配，界面不变形
- 交互反馈：拖拽文件有高亮效果，按钮点击有动画，操作状态清晰可见

## 🚀 怎么用？（3步搞定）

### 第一步：安装软件
- **简单方式**：下载exe文件，双击运行就行（建议右键"管理员运行"）
- **技术方式**：有Python环境的可以下载源码运行

### 第二步：启动服务
- 运行后看到托盘有个小图标就说明成功了
- 软件会显示一个网址，比如：http://192.168.1.100:5000
- 如果有问题，软件会告诉你缺什么，按提示下载安装就行

### 第三步：开始打印
- 用手机或其他设备打开刚才的网址
- 选择打印机和打印设置
- 把文件拖进去或点击选择文件
- 点"上传并打印"就完事了！

### 💡 小贴士
- **托盘图标很有用**：左键打开网页，右键有各种设置
- **端口被占用了**：右键托盘图标→服务设置→更改端口，保存后需要手动重新运行程序
- **想要开机自启**：右键托盘图标→开机自启→开启
- **网络有问题**：右键托盘图标→网络配置→按需设置

## ❓ 遇到问题怎么办？

### 软件打不开？
**症状**：双击exe没反应，或者提示缺少dll文件  
**解决**：下载安装VC++运行库（软件会提示下载地址），安装后重启电脑

### 找不到打印机？
**症状**：网页里显示"未检测到可用打印机"  
**解决**：
- 检查打印机是否开机、连接正常
- 确认打印机驱动已安装
- 点网页里的"刷新"按钮重新检测

### 网页打不开？
**症状**：输入地址后无法访问  
**解决**：
- 检查Windows防火墙，允许程序通过
- 尝试用127.0.0.1替代IP地址
- 右键托盘图标→更改端口，换个端口后手动重新运行程序

### 杀毒软件报毒？
**症状**：杀毒软件阻止运行  
**解决**：添加到白名单，或者从官方GitHub下载最新版本

### 更改端口后程序没反应？
**症状**：更改端口后程序还是使用旧端口  
**解决**：端口配置已保存，但需要手动重新运行程序才能生效。退出程序后重新双击exe文件启动即可

### 点击选择文件没反应？
**症状**：点击上传区域没有弹出文件选择框  
**解决**：
- 更新浏览器到最新版本
- 清除浏览器缓存
- 关闭广告拦截插件
- 直接拖拽文件到上传区域

### 想要横版打印或自定义页码？
**现实**：这个软件主要解决"能打印"的问题，高级排版功能有限  
**建议**：用Word、WPS等软件先排好版，设好参数，再上传打印
