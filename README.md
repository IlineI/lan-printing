# 内网打印服务 1.6版本
- 使用 Python3.7.9 编程和打包，兼容性更好

## 主要功能
- 【部署简单】：只需在连打印机的电脑上安装本软件，同一个局域网下的其他设备无需安装任何驱动或客户端，只需在浏览器输入 IP:端口 即可调用目标电脑的打印机进行打印
- 【支持多系统多用户】：局域网内多用户共享打印，支持 Win7/Win10/Win11
- 【支持跨平台访问】：支持Windows、Android、iOS等各种设备访问
- 【支持静默打印】：无需弹窗、无需确认，自动发送到目标打印机
- 【文件上传与预览】：支持单文件选择、多文件选择和拖拽上传三种方式，图片和PDF文件上传后支持预览
- 【打印队列管理】：实时监控Windows打印队列状态，支持任务取消
- 【支持多种文件格式】：JPG/PNG、PDF、DOC/DOCX、XLS/XLSX、PPT/PPTX、TXT等（建议用JPG和PNG，其他格式不推荐）
- 【打印参数自定义】：份数、单双面、纸张大小、打印质量
- 【智能边距】：自动检测页眉页脚内容，无页眉页脚时最小化边距，最大化打印内容显示区域
- 【过滤虚拟打印机】：自动检测并过滤虚拟打印机
- 【打印日志记录】：显示打印用户的计算机名/设备型号、时间、参数、IP地址
- 【打印日志自动清理】：每天下午3点自动清理7天前记录，防止日志文件过大
- 【托盘栏图标集成功能】：网络配置（静态IP/DHCP）、端口修改、日志管理、控制台隐藏/显示、开机自启等
- 【健康检测和重启服务】：健康检测与自动重启，保证服务稳定运行
- 【控制台窗口显示/隐藏】：源码运行或exe运行均支持控制台窗口显示/隐藏（可查看报错信息）
- 【控制台信息自动清理】：避免控制台长时间运行内容过多卡死
- 【完美退出】：自动清理残留进程
- 【支持内网穿透】：自动识别内网/公网/内网穿透模式，兼容各种内网穿透工具，支持公网访问
- 【文件文件兼容性优化】：优化文本文件UTF-8/GBK/GB2312等编码格式的兼容性


## 1.6版本更新内容
- 页面边距智能调整，解决打印内容偏小问题，保留用户设置
- 将IP配置和端口配置合并到【网络配置】菜单，操作更简洁直观
- 简化为"设置当前IP为静态"功能，自动推导网关地址，避免复杂的手动配置
- 完善对Microsoft Office和WPS Office的COM对象控制，提供更可靠的打印边距设置
- 添加控制台自动清理功能


## 使用说明
- 右键exe文件，使用管理员权限打开，托盘栏会生成图标
- 右键图标，会展示IP端口，在浏览器输入即可访问
（可用二维码生成工具，做成二维码分发给局域网的其他人使用）
- 打开网页，选择/拖拽文件上传，设置打印参数后提交即可
- 上传后可在队列中删除未打印的文件，随时取消
- 所有操作均有提示，界面直观，无需额外说明
- 网络相关设置请在托盘菜单【网络配置】中操作（必须管理员权限）
- 支持静态IP设置：选择"设置当前IP为静态"可固定当前IP地址
- 智能打印优化：程序会自动优化文档边距，无需手动调整
- 日志自动维护，无需手动清理
- 控制台自动维护，无需手动清理

## 修复工具
  - 若 `print_server1.6.exe` 运行报错，可进入 `exe` 文件夹使用对应系统的修复工具
  - `win7_fix_tool.exe` - Windows 7 专用修复工具
  - `win10_fix_tool.exe` - Windows 10 专用修复工具  
  - `win11_fix_tool.exe` - Windows 11 专用修复工具

## 系统要求与兼容性
- 操作系统：Windows 7/8/10/11（32位/64位）
- 内存要求：建议 2GB 以上
- 磁盘空间：至少 100MB 可用空间
- 网络要求：支持TCP/IP协议的网络环境
- 打印机：支持Windows标准打印接口的所有打印机
- 浏览器：支持现代Web标准的浏览器（Chrome、Firefox、Edge、Safari等）
- Office软件：支持Microsoft Office 2013/2016/2019/365、WPS Office等（用于文档打印优化）
- 网络权限：建议以管理员权限运行，以获得完整的网络配置和打印机控制功能
