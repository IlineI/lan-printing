1.2版本功能说明：

1. 在A电脑运行exe文件，会在任务栏生成一个托盘图标。
2. 托盘图标上会显示本机IP和端口。
3. 支持开机自启，可在托盘菜单一键切换。
4. 自动识别所有本地和网络连接的打印机（支持主流品牌和虚拟打印机）。
5. B电脑或手机（同一局域网）在浏览器输入A电脑IP:端口，即可访问打印服务。
6. 支持上传PDF/JPG/PNG/TXT文件，支持多文件同时上传。
7. 支持选择打印机、份数、单双面、纸张大小、纸张质量等参数。
8. 物理打印机支持静默打印，无需A电脑手动确认。
9. 虚拟打印机（如Microsoft Print to PDF）打印时会弹出保存窗口，需A电脑手动选择保存路径，无法完全自动化。
10. 打印日志可在网页端查看最近10条。
11. 所有上传文件统一保存到A电脑桌面的“lan-printing-uploads”文件夹，10分钟后自动清理（后台线程实现）。
12. 打印异常或文件名冲突等错误会在网页端弹窗提示。
13. 文件预览支持图片、PDF、TXT，文件不存在时有友好提示。
14. 支持Win7/Win10系统，打包后可直接运行，无需Python环境。
15. PDF文件无需安装Adobe Reader，统一用Windows系统自带命令打印。
