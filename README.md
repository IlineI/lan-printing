# 内网打印服务 1.5版本

## 主要功能
- 局域网内多用户共享打印，支持 Win7/Win10/Win11
- 支持静默打印：无需弹窗、无需确认，自动发送到目标打印机
- 支持多文件批量上传、拖拽上传，图片和PDF文件上传后支持预览
- 打印队列管理：可随时删除未打印文件，灵活取消
- 支持多种文件格式：图片、PDF、Word、Excel、文本等
- 打印参数自定义：份数、单双面、纸张大小、打印质量
- 自动检测并过滤虚拟打印机
- 打印日志记录：显示打印用户的计算机名/设备型号、时间、参数
- 打印日志自动清理：每天下午3点自动清理，防止日志文件过大
- 托盘栏图标集成功能：网络配置、日志管理、服务设置、开机自启
- 网络配置（静态IP/DHCP）和端口修改，全部集成在托盘菜单
- 健康检测与自动重启，保证服务稳定运行
- 支持控制台窗口显示/隐藏
- 完美退出，自动清理残留进程
- 内网穿透兼容优化
- 文本文件编码自动识别，兼容更多格式
- 部署简单：只需在连打印机的电脑上安装本软件，其他设备无需安装任何驱动或客户端，其他设备只需在浏览器输入IP:端口即可调用目标电脑的打印机进行打印

## 1.5版本更新内容
- 使用 Python3.7.9 编程和打包，兼容性更好
- 网络配置功能集成在托盘栏右键菜单
- 取消实例检测和环境检测
- 修复长时间运行卡死问题
- 修复双面打印功能
- 修复下拉菜单波浪线问题
- 修复退出功能，能够完美退出
- 优化文本文件兼容性，支持更多编码格式
- 优化内网穿透兼容性
- 打印日志显示计算机名称/设备名称
- 新增打印日志自动清理功能，每天下午3点清理

## 使用说明
- 打开网页，选择/拖拽文件上传，设置打印参数后提交即可
- 上传后可在队列中删除未打印的文件，随时取消
- 所有操作均有提示，界面直观，无需额外说明
- 网络相关设置请在托盘菜单中操作
- 日志自动维护，无需手动清理
