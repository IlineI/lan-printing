# 内网打印服务 v1.3

## 功能说明

### 1. 打印管理功能
1. **多格式文件支持** - 支持PDF、JPG/PNG图片、DOC/DOCX/PPT/PPTX/XLS/XLSX Office文档、HTML/HTM网页、TXT文本文件的静默打印
2. **智能打印机识别** - 自动过滤虚拟打印机（如WPS PDF、Microsoft Print to PDF、导出为WPS PDF等），仅显示真正的物理打印机
3. **高级打印设置** - 支持打印份数设置（1-10份）、单双面打印选择、纸张大小选择（A4/A3/Letter/Legal）、打印质量设置（普通/高质量/相片纸）
4. **静默打印技术** - 无需手动确认，文件自动发送到指定打印机，支持多种打印引擎（Adobe Reader、Office COM对象、Windows图片查看器、PowerShell COM等）
5. **多文件批量打印** - 支持同时上传多个文件进行批量打印处理
6. **打印队列管理** - 显示待打印文件列表，支持文件预览功能（PDF、图片、文本）
7. **详细打印日志** - 记录所有打印操作的时间、文件名、打印机、设置参数等信息，保留最近10条记录
8. **虚拟打印机警告** - 智能识别并警告用户选择的虚拟打印机，避免误操作

### 2. 网络配置功能
9. **网络状态监控** - 实时显示当前IP地址、子网掩码、默认网关、网络适配器等信息
10. **静态IP设置** - 支持手动设置固定IP地址，包括IP、子网掩码、网关的自定义配置，支持自动推导网关
11. **DHCP自动获取** - 支持启用DHCP动态获取IP地址，一键切换网络模式
12. **智能IP建议** - 根据当前网段自动推荐合适的静态IP地址（通常为当前网段.100）
13. **网络适配器管理** - 自动识别网络适配器，支持多种网卡类型，兼容本地连接和以太网命名
14. **网络容错机制** - 支持多种IP获取方式，断网情况下程序仍能正常运行

### 3. 系统集成功能
15. **系统托盘服务** - 后台运行，通过系统托盘图标管理服务状态，支持🖨️表情图标
16. **开机自启动** - 支持设置开机自动启动，通过注册表管理启动项
17. **Web界面管理** - 基于Bootstrap 5.3的响应式Web界面，支持移动设备访问，离线CSS备用
18. **局域网访问** - 支持局域网内其他设备通过IP地址访问打印服务（默认端口5000）
19. **文件自动清理** - 定期清理上传的临时文件（10分钟过期），避免磁盘空间占用
20. **资源路径兼容** - 兼容PyInstaller打包和源码运行，支持相对路径和绝对路径

### 4. 用户体验功能
21. **多标签页界面** - 打印管理和网络配置分离显示，界面简洁易用，支持标签页切换
22. **实时状态反馈** - 打印成功/失败状态实时显示，操作结果及时反馈，使用✅❌图标
23. **详细操作说明** - 内置完整的功能说明和使用指导，包含静默打印说明
24. **错误处理机制** - 多重备用打印方案，确保打印成功率，包含ShellExecute、COM对象、批处理等
25. **中文本地化** - 完全中文化界面，符合国内用户使用习惯
26. **离线运行支持** - 网络断开时仍能正常使用，自动降级到离线CSS和JavaScript

### 5. 高级功能
27. **托盘右键菜单** - 丰富的系统托盘菜单，包含服务地址、IP状态、网络配置等快捷操作
28. **智能图标备用** - logo.ico文件缺失时自动使用🖨️表情图标，支持多种字体备用
29. **打印机能力检测** - 自动检测打印机双面、彩色等功能支持
30. **文件类型验证** - 严格的文件类型检查，支持的格式包括PDF、Office文档、图片、HTML、文本
31. **批量打印优化** - 针对不同文件类型使用最优的打印方案，提高成功率
32. **COM对象集成** - 深度集成Office COM对象，支持Word、Excel、PowerPoint文档的精确打印

## 技术特性
- **网络容错性** - 多层网络检测机制，支持外网不通、内网断开等各种网络状态
- **跨平台兼容** - 基于Python Flask框架，主要针对Windows系统优化
- **多打印引擎** - 集成Adobe Reader、Office应用程序、Windows原生打印、PowerShell COM对象等多种打印引擎
- **智能备份机制** - 主打印方案失败时自动切换备用方案，确保打印成功
- **WMI系统调用** - 使用Windows WMI接口进行网络配置管理和系统信息获取
- **离线CSS/JS** - 内置完整的Bootstrap样式备用，CDN失败时自动切换离线版本
- **多字体支持** - 系统托盘图标支持表情字体、Arial、Calibri等多种字体备用
- **异常恢复** - 全面的异常处理机制，程序在各种错误情况下都能稳定运行

## 运行要求
- **操作系统**: Windows 7/8/10/11
- **Python版本**: 3.6+
- **依赖库**: Flask, win32print, win32api, pystray, PIL等
- **权限要求**: 网络配置功能需要管理员权限
